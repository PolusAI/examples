# Streamlit Stroke Predictor Dashboard

## Dashboard Description
This dashboard includes a stroke predictor form containing fields for determinants of stroke. A user can provide this data for a hypothetical individual, which are then used by a classifier to predict the likelihood of the individual experiencing stroke.

## Resources 
* Dataset provided by Kaggle at https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset. 
* Learn more about Streamlit at https://streamlit.io/.
